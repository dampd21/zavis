# 아우라비즈 데모 페이지

이 데모는 아우라비즈 웹사이트의 메인 페이지 UI를 재현한 정적 HTML 페이지입니다.

## 📁 구조

- `index.html`: 메인 페이지
- `css/style.css`: 커스텀 스타일
- `js/main.js`: JavaScript 및 Swiper/ECharts 초기화

## 🧪 기술 스택

- HTML5
- CSS3
- Bootstrap 5
- Swiper.js
- ECharts
- Vanilla JS

## 🚀 실행 방법

브라우저에서 `index.html` 파일을 열어 확인할 수 있습니다.
